# 🧑‍💻 조설훈의 웹 개발 포트폴리오

안녕하세요! 개발자 지망생 **조설훈**의 개인 웹 포트폴리오입니다.  
프론트엔드와 백엔드를 함께 공부하며 성장해 나가는 과정을 담고 있습니다.

## 📂 프로젝트 소개

이 웹페이지는 HTML, CSS, JavaScript로 제작된 **반응형 포트폴리오 웹사이트**입니다.  
간단한 탐색 메뉴, 부드러운 스크롤, 다크 모드 기능이 포함되어 있습니다.

## 🌟 주요 기능

- ✅ 소개, 프로젝트, 연락처 섹션
- ✅ 탐색 메뉴에서 현재 위치 자동 강조
- ✅ 부드러운 스크롤 이동
- ✅ **다크 모드 / 라이트 모드** 전환 버튼
- ✅ 반응형 레이아웃

## 🛠 사용 기술

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
